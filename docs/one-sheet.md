

# **RaidGuild Cohort x402 Facilitator**

## *Turn APIs into revenue streams. Bring devs into the RaidGuild orbit.*

### **What it is**

A **RaidGuild-branded x402 facilitator** that lets developers charge micro-payments (fractions of a cent) for API calls — instantly, trustlessly, and without signup friction. Think of it as a **toll gate** you can drop in front of any endpoint.

For the broader ecosystem, it’s a **public utility** with strong RaidGuild flavor.
For us, it’s a **brand funnel**, a **business development engine**, and a **strategic foothold** in the fast-growing agentic wallet + micropayments landscape.
